CrownHackHour
=============

A repository for the Hack Hour at Crown

Getting Started
===============

* Install Chocolatey (https://chocolatey.org/)
* `choco install nodejs`
* Make sure npm is in your path with `where npm`
* `npm install -g jade`
* Ensure jade is in your path with `where jade`
